## Programación Para Dispositivos Móviles  

# **Con-tacto**

Aplicación móvil tipo agenda de contactos que permite iniciar sesión con Google, agregar, eliminar, editar y bloquear contactos. También permite agrupar contactos mediante etiquetas, y generar códigos QR tanto para el perfil del usuario como para compartir cada contacto individualmente.

##  Instalación (Android Studio)

## 1️.  Clonar el repositorio
```bash
git clone https://github.com/marquezj30/FINAL-PROGRAMCION-MOVIL.git

```
## 2️.  Abrir el proyecto en Android Studio
- Abrir Android Studio  
- Seleccionar **Open an Existing Project**  
- Elegir la carpeta del proyecto clonado  

## 3️.  Sincronizar el proyecto
- Esperar que Android Studio sincronice Gradle automáticamente  

## 4️.  Seleccionar dispositivo
- Conectar un emulador Android  

## 5️.  Ejecutar
- Presionar Run  en Android Studio  
- Esperar a que compile y se abra la app  

##  Descripción del Proyecto
Con-tacto es una agenda digital moderna que permite al usuario:
- Iniciar sesión mediante Google  
- Crear, editar, eliminar y bloquear contactos  
- Organizar contactos en grupos con etiquetas  
- Ver su perfil y compartirlo mediante un código QR  
- Compartir contactos usando QR individual  
- Navegar con una barra inferior (Contactos – Grupos – Perfil)
- Busqueda por nombre y apellido   
- Cerrar sesión
